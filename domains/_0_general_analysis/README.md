## Tools

| Tool | What it does |
|------|--------------|
| `list_tables` | List tables in a dataset |
| `describe_table` | Show table schema |
| `sample_table` | Quick row preview |
| `query_bigquery` | Run SQL |
| `profile_table` | Data quality check (nulls, distinct counts) |

## Examples

```
"list tables in prod-im-data.mod_imx"
"sample 10 rows from hubspot_b2b_deal"
"how many deals closed last month?"
"profile the deals table"
```
